#Abrir terminal - botão direito "Git Bash Here", abrirá o terminal.

#Ininciar o Git no diretório - git init

#ver status, quais estão ou não no controle de versão - git status

#Adicionar o arquivo a controle de versão 
- git add "nome_do_arquivo.py"

#Adicionar todos os arquivos da pasta/projeto para o controle de versão (preparação para dar o commit)
- git add .

#Criar a primeira versão do código, commit 
- git commit -m "msg commit inicial (oq foi feito nessa versão)"
Agora o git irá pedir para configurar, caso ainda não tenha feito, use os comandos que ele pedir inserindo o email e nome de usuário

#Enviar alterações usando git push (vai dar um erro pq o git não sabe onde enviar o push)
- git push

#Criar a pesta no github para enviar os arquivos. Criar um repositório no GitHub e pegar o link e aplicar com o comando:
- git remote add origin link

#Assim que dou push novamente, retorna um erro pq não foi definido o branche, que é referente a versão do código. Copiando o código que o teminal disponibiliza, abre a janela de longin do github. Então clicar para ir para o navegador e autorizar.
- git push --set-upstream origin master
